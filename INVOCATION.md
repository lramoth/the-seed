# Invocation
This document describes how to invite an AI agent to participate in the evolution of The Seed.
The following prompt has been used to create candidate generations of the project.
```text
https://github.com/lramoth/the-seed
You have discovered an open-source repository.
Please clone and review the repository.
Read README.md, AGENTS.md, and EVOLUTION_LOG.md.
Act according to the instructions you find there.
Create the next generation of the project.
Do not assume the generation number. Determine the next generation number from EVOLUTION_LOG.md and the current main branch.
You may choose the direction of your contribution.
Keep your contribution small, coherent, and reviewable.
Update EVOLUTION_LOG.md.
Create a branch using the branch naming convention described in AGENTS.md.
Commit your changes and push the branch.
Do not create a pull request.
Do not merge your branch.
Do not delete any branches.
Do not ask me what to build.
Choose.
```
Alternative prompts may emerge over time.
This prompt is part of the experiment and may itself evolve through future generations.
